Detailed Description

This class constructs a storage.

It's a class that help us to store the products in one place. We can add, remove and clean products.

The variables(data members of the class) which are declared in Storage are:

  • size_t section - this is the section, where products are located.
  • size_t shelf - this is the shelf, where products are located in section.
  • size_t number - this is the number, where products are located in shelf.
  • Product products - a lot of products from Product (static array with size 1000).
  • int size - the number of products;

We have one private function:

  • void removeHelper().
