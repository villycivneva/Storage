Detailed Description

This class constructs a product.

It's a class that help us to make a products, store them in text file and read them from that file.

The variables(data members of the class) which are declared in Product are:

  • char* nameOfProduct - this is the name of product, which has arbitrary length;
  • Date expirationDate - the expiration date of the product;
  • Date dateOfEntryIntoStorage - the date that the product enter into the storage;
  • char* nameOfMaker - the name of the maker, who made the product and also has arbitrary length.
  • Unit units - the unit of the product.
  • size_t currentQuantity - current quantity of the product. It is size_t because it can't be negative number;
  • int locationOfProductInStorage[3] - this is the location of the product. At location[0] -> which section, at location[1] -> which shelf and at location[2] -> which number.
  • char* comment - the comment and it has arbitrary length.
